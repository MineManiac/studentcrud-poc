# Student CRUD POC

Student registration CRUD application.

The project uses Node.js, Express, MongoDB, React, and Vite, with a simple web interface to create, edit, delete, and list students.

## Features

- Student registration.
- Automatic sequential code generated by the backend.
- Student editing by clicking a row in the grid.
- Student deletion with confirmation.
- Grid listing.
- The most recently inserted student appears at the top by default.
- Clickable grid sorting by:
  - Code.
  - Birth date.
  - Registration date.
  - Last update.
- Automatic address lookup by ZIP code using ViaCEP.
- ZIP code input mask.
- Address fields are automatically filled when the ZIP code exists.
- Registration date and last update are shown only in the grid.
- Sensitive variables are protected through `.env`.

## Student Fields

- Automatic code.
- Name.
- Birth date.
- ZIP code.
- Street.
- Number.
- Address complement.
- District.
- City.
- State.
- Registration date.
- Last update.

## Validations

The main validations run on the backend to prevent invalid data from being saved even if someone bypasses the interface.

- Name is required.
- Birth date is required.
- Birth date must be a real date.
- Birth date must be between `01/01/1900` and the current date.
- ZIP code is required.
- ZIP code must have 8 digits.
- ZIP code must exist in ViaCEP.
- State must be a valid Brazilian state abbreviation.
- The provided address must be compatible with the ZIP code returned by ViaCEP.
- Street, number, district, city, and state are required.

## How to Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create the `.env` file based on `.env.example`:

   ```bash
   cp .env.example .env
   ```

   On Windows PowerShell, you can use:

   ```powershell
   Copy-Item .env.example .env
   ```

3. Configure the MongoDB connection in `.env`.

   Local MongoDB example:

   ```env
   MONGODB_URI=mongodb://127.0.0.1:27017/studentcrud
   PORT=3000
   ```

   MongoDB Atlas example:

   ```env
   MONGODB_URI=mongodb+srv://user:password@cluster.mongodb.net/studentcrud?retryWrites=true&w=majority
   PORT=3000
   ```

4. Run the React frontend and Express backend in development mode:

   ```bash
   npm run dev
   ```

5. Open the React app in the browser:

   ```text
   http://localhost:5173
   ```

## Scripts

- `npm start`: starts the Express application and serves the React production build.
- `npm run dev`: starts the Express backend and React/Vite frontend together.
- `npm run dev:server`: starts only the Express backend.
- `npm run dev:client`: starts only the React/Vite frontend.
- `npm run build`: creates the React production build.
- `npm run preview`: previews the React production build through Vite.

## API Routes

- `GET /api/health`: checks if the API is responding.
- `GET /api/students`: lists students.
- `POST /api/students`: creates a student.
- `PUT /api/students/:id`: updates a student.
- `DELETE /api/students/:id`: deletes a student.
